## Overview

This repository documents **hands-on AWS cloud security projects** focused on hardening, monitoring, and auditing cloud infrastructure. Each project applies security best practices aligned with the **AWS Well-Architected Framework (Security Pillar)**, **CIS AWS Benchmarks**, and **NIST SP 800-144**.

**Certifications Applied:** AWS Certified Cloud Practitioner (2026)
**Target Role:** Cloud Security Engineer, Junior Cloud Security Analyst, SecOps Engineer

## Security Pillars Covered

- Identity & Access Management (IAM) - Least privilege, MFA, role-based access
- Infrastructure Protection - VPC segmentation, security groups, NACLs
- Data Protection - S3 encryption, KMS, Macie classification
- Detection & Monitoring - CloudTrail, CloudWatch Alarms, GuardDuty
- Incident Response - Automated response with Lambda, SNS
